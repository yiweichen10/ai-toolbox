MiniMax Token Plan发布+Seedance 2.0上线+AI新贵市值超百度：2026年3月第4周AI圈发生了什么

2026年3月25日 · AI视频

前言

2026年3月的最后一周，AI圈连续爆了三个大新闻。MiniMax把编程订阅计划升级为覆盖所有AI模态的Token Plan，字节Seedance 2.0在视频生成评测中超过了Sora和Veo 3，而MiniMax在港交所上市才61天，市值已经冲过百度。

三个事件指向同一个趋势：国产AI不再追赶，开始定义新的游戏规则。

事件一：MiniMax Token Plan——一个订阅用所有AI模态

发生了什么

3月23日，MiniMax宣布将其原有的编程订阅计划Coding Plan全面升级为Token Plan，官方宣称这是全球首个覆盖全模态模型的统一订阅计划。

简单说：以前你用ChatGPT对话付一份钱、用Midjourney画图付一份钱、用Suno做音乐又付一份钱。现在MiniMax说，¥49/月，编程、视频、语音、音乐、图像全部打包给你。

对用户意味着什么

创作者：如果你是自媒体运营者，需要同时用AI写文案、做配图、配音、做短视频、生成BGM，Token Plan一口气全包了。¥49/月覆盖了之前需要5个工具订阅才能完成的工作流。

开发者：M2.7是MiniMax的主力编程模型，主打"自我进化"和"Cowork Agent"，在SWE-bench上的表现紧追[Claude Code](/tools/claude-code/)。加上视频和语音能力，对需要做AI Agent演示的开发者来说非常实用。

我的判断：¥49/月的价格策略明显是在打价格战。单看任何一个模态，MiniMax可能都不是最强的——对话不如ChatGPT、画图不如Midjourney、视频刚被Seedance超过——但"全包"这个概念本身就有杀伤力。对于预算有限但需要多模态能力的用户，这是目前性价比最高的方案。

事件二：字节Seedance 2.0力压Sora，登顶视频生成榜

发生了什么

字节跳动的视频生成模型Seedance 2.0正式上线，在Artificial Analysis的Text-to-Video盲测中拿到Elo 1269分，排名第一，超过了OpenAI的Sora和Google的Veo 3。尤其在Image-to-Video（图生视频）赛道，Seedance 2.0被评价为具有"压倒性优势"。

Seedance 2.0的核心能力

能力说明
多模态输入支持文本、图像、音频、视频混合输入
音画同步生成画面时同步计算音频，口型自然对齐
物理模拟运动稳定性高，物体运动符合物理规律
镜头控制支持推、拉、摇、移专业镜头运动
输出分辨率1080p

原生音画同步是Seedance 2.0最大的卖点。之前AI生成视频最大的痛点就是口型和声音对不上，需要后期用ElevenLabs等工具重新配音。Seedance直接在生成阶段就解决了这个问题，对短视频创作者来说是实打实的效率提升。

可用平台

  • 国际版：Dreamina网页端（dreamina.capcut.com）
  • 国内：剪映桌面版和移动版已集成

注意，Seedance 2.0已经集成到剪映里了——这意味着你不需要单独开一个网页去用AI生成视频，直接在剪映的编辑流程里就能调用。这种"嵌入工作流"的思路和Cursor嵌入编辑器的思路一样，降低了使用门槛。

对行业的影响

Sora是OpenAI的王牌视频模型，Veo 3是Google最强视频生成方案。Seedance 2.0超过它们，说明国产AI在视频生成领域已经不落后了——甚至在某些维度（音画同步、图生视频）已经领先。

对用户来说更实际的问题是：你不再需要翻墙去用Sora，剪映里直接就能用Seedance。如果你主要做短视频内容，Seedance 2.0+剪映的组合可能是目前最顺手的方案。

事件三：MiniMax上市61天，市值超百度

发生了什么

3月17日，MiniMax（0100.HK）市值突破3000亿港元，正式超过百度的3322亿港元。3月23日有报道确认，MiniMax上市仅61天，市值稳定在3000亿港元上方。

时间线

时间事件

2026年3月10日股价单日涨超22%，市值首次超百度（3826亿港元）
2026年3月17日市值稳定突破3000亿港元
2026年3月23日确认上市61天市值超百度

背后的意义

MiniMax成立于2022年，创始人是前商汤科技实习生闫俊杰（河南县城出身）。从800亿港元到3000亿港元，用了不到两个月。百度从搜索巨头走到被一家4年历史的AI公司超越，这个转折比任何数据都更有冲击力。

资本市场的投票说明了一件事：纯AI公司比"AI+"公司更被看好。 百度也有文心一言，也有AI布局，但市场给百度的估值逻辑还是"搜索引擎+广告公司"。而MiniMax从第一天起就是纯AI公司，估值逻辑完全不同。

值得关注的另一个信号：MiniMax和智谱在2026年1月先后登陆港交所，中国AI公司上市潮已经开始。这轮上市的公司不是PPT故事，而是有实际产品（海螺视频用户超2亿）和营收支撑的——MiniMax首份财报显示营收翻倍增长。

三件事放在一起看：国产AI的商业化加速

把三个事件串起来看，能发现一个清晰的脉络：

  • MiniMax Token Plan：国产AI开始用价格战抢占用户，全模态打包策略是"先用低价把人拉进来"
  • Seedance 2.0：技术能力不再是短板，在具体场景（视频生成）已经超过国际竞品
  • 市值超百度：资本市场认可国产AI的商业价值，资金持续流入

对普通用户来说，最直接的利好是：选择变多了，价格变低了，不用翻墙也能用一流工具了。

如果你之前一直在纠结要不要花$20/月订阅ChatGPT Plus，现在有了更多选择：MiniMax Token Plan ¥49/月覆盖全模态、Kimi完全免费200万字上下文、豆包免费集成了Seedance。对于中文用户，2026年的AI工具生态已经非常成熟。

FAQ

Q1：MiniMax Token Plan的Plus版¥49/月值得订阅吗？ A：取决于你的使用需求。如果你只用文本对话，¥49偏贵——Kimi免费、ChatGPT免费版够用。但如果你需要同时用AI做视频、音乐、语音和图像，¥49覆盖全模态是目前市场上最便宜的全包方案。建议先试Basic版¥29/月体验M2.7编程能力，满意再升级Plus。

Q2：Seedance 2.0免费吗？ A：Seedance 2.0已集成在剪映中，支持免费试用和付费订阅。具体定价需要打开剪映查看最新方案。对比Sora（需ChatGPT Plus $20/月），Seedance的获取门槛明显更低。

Q3：MiniMax市值超过百度意味着什么？ A：这是资本市场对"AI native"公司的定价信号。MiniMax没有搜索引擎、没有广告业务、没有App矩阵，纯靠AI产品（海螺视频、talkie等）和API服务做到了3000亿市值。对行业意味着：AI不再是传统互联网公司的附属品，而是新的核心资产。

Q4：普通用户如何从这些变化中受益？ A：最直接的受益是工具选择更多、价格更低。之前需要翻墙+付费的工具，现在国内有免费或低价的替代方案。建议关注：MiniMax Token Plan（全模态打包）、剪映里的Seedance 2.0（AI视频生成）、Kimi（免费长文档分析）、豆包（免费日常助手）。

Q5：国产AI在技术上真的超过国外了吗？ A：在具体赛道上，国产AI已经开始超越或追平。Seedance 2.0超过Sora是视频生成领域的例子；DeepSeek登顶全球AI应用百强榜第四；Claude Code用了Kimi 2.5作为基座模型。但在通用推理、多模态理解等综合能力上，GPT-5和Claude 4仍有优势。更准确的说法是：差距在缩小，在特定场景已经反超。

总结

2026年3月最后一周的AI圈用一个词概括：弯道超车。 MiniMax用全模态订阅重新定义了AI工具的定价逻辑，Seedance 2.0用实际评测成绩证明了国产视频生成能力，而资本市场用3000亿港元为这场变革投了赞成票。

对用户来说，最实用的建议：别急着订阅昂贵的海外工具，先试试MiniMax Token Plan和剪映的Seedance 2.0。2026年的国产AI工具，已经值得认真对待了。
