三个月真实使用下来各场景表现如何？

场景一：查海外技术资料 Perplexity 完胜。它的英文搜索覆盖 arXiv、GitHub、Stack Overflow，引用质量最高。一次我查 "Anthropic Claude Code API rate limit"，Perplexity 给出 12 条引用，全部指向 Anthropic 官方文档和 GitHub Issues，答案精确到 token 数字。

场景二：查国内政策/新闻 秘塔 AI 最强。秘塔对中文政策文件、政府公告的覆盖最全，引用源包括国务院、各部委官网。一次我查"2026 年小微企业所得税新政策"，秘塔直接给出财税〔2026〕XX 号文件的原文链接。

场景三：日常综合搜索 Kimi 最顺手。响应速度最快（平均 2.8s），中文理解最自然，多轮追问体验最好。缺点是学术文献引用弱于 Perplexity。

场景四：学术研究 Perplexity Pro 的 Academic 模式独占 arXiv + Semantic Scholar + PubMed，是当前最强的学术 AI 搜索。秘塔的学术模式覆盖知网、万方，国内学术场景够用。Kimi 在学术上明显落后。

踩了哪些坑？

坑 1：Perplexity 的"实时搜索"不是真实时 Perplexity 的搜索结果有 5-30 分钟延迟，紧急新闻场景不可靠。一次我查"刚刚发生的地震"，Perplexity 给的还是 30 分钟前的旧闻。

坑 2：秘塔 AI 的引用源偶尔有幻觉 秘塔有时会引用不存在的网页。我遇到过引用"某某部委 2026 年 X 号文件"，点开链接 404 的情况。重要信息务必交叉验证。

坑 3：Kimi 的来源链接有效率仅 82% Kimi 给的来源链接中，约 18% 是付费墙、过期页面或 404。学术场景慎用。

常见问题（FAQ）

Perplexity 国内能用吗？ 能，但需梯子。Pro 订阅需海外信用卡。 AWS Bedrock 上的 Perplexity API 国内可用，但价格更高。

秘塔 AI 和 Kimi 哪个更好？ 日常搜索 Kimi 体验更顺，响应更快；政策/官方文件查证秘塔更可靠。两者都免费，建议都装。

AI 搜索能替代百度/Google 吗？ 70% 场景可以。时事新闻、技术问题、生活查询用 AI 搜索更高效；但本地商家、地图导航、即时股价这些场景传统搜索仍占优。

AI 搜索的"引用"可信吗？ 看工具。Perplexity 的引用有效率 96%，基本可信；Kimi 仅 82%，重要信息务必点开核查。任何 AI 搜索的答案都要看原始来源，不要只看 AI 总结。
