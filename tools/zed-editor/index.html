Zed Editor是什么？

如果你最近在关注代码编辑器的动静，应该听说过Zed Editor这个名字。它是Atom编辑器的原班人马搞的新项目，用Rust重写的，主打的就是一个“快”字。

说白了，Zed就是一个面向开发者的代码编辑器，2025年还推出了自己的AI功能——Zed AI。它想解决的问题很简单：让写代码这件事更流畅、更高效，同时把AI能力直接集成进编辑器里，而不是让你来回切换工具。

核心功能

1. Zed AI：内置AI编程助手

这个应该是Zed最被关注的功能了。它集成了Claude、Gemini等多个主流大模型，你可以直接在编辑器里问AI问题、让它帮你补全代码、解释你选中的代码段。不用再开一个浏览器标签页去ChatGPT，集成度很高。不过要注意，AI功能需要Pro版本才能使用，免费版没有这个待遇。

2. 协作模式：实时共同编辑

Zed支持实时共享编辑器和终端，类似于Google Docs的协作体验。你可以把编辑窗口分享给队友，双方能看到对方的光标和操作。这个功能对远程协作、代码审查、教学场景都挺实用的。

3. 多光标编辑

这是现代编辑器的标配功能了，Zed做得也不错。你可以同时在多个位置插入光标，进行批量修改。处理重复性代码时很省力，掌握之后效率提升明显。

4. Git集成 + 终端模拟器

Zed内置了完整的Git支持，提交、分支、查看历史这些操作都可以在编辑器里完成。同时它还集成了终端，不用另外开窗口，直接在编辑器里就能跑命令。这两个功能组合起来，日常开发基本不需要频繁切换工具。

5. 远程开发

支持连接远程服务器开发，对于需要在服务器上直接改代码或者使用远程算力的场景，这个功能很实用。

值不值得用？

优点：

  • 速度确实快，响应延迟很低，用起来很流畅
  • AI补全和对话集成做得比较自然，不用来回切换
  • 界面清爽，主题可定制
  • 协作功能对团队使用场景很友好

缺点：

  • 插件生态目前不如VS Code丰富，有些常用插件可能还没有
  • Windows版的稳定性略逊于macOS版
  • AI功能要付费才能用
  • 企业级功能（比如SSO）还在完善中

我的看法： 如果你追求编辑器的流畅度，对AI辅助编程有需求，Zed是一个值得尝试的选择。特别是如果你已经用腻了VS Code，想换个轻量点的工具，它不会让你失望。但如果你特别依赖VS Code那些插件，或者在Windows上使用且追求稳定性，可能需要观望一下，等生态更成熟再用。

使用建议

  • 先试用免费版：把基础功能摸熟，看看编辑体验是否符合你的习惯
  • 熟悉快捷键：Zed的快捷键设计和一些主流编辑器不太一样，上手需要适应一下
  • 如果需要AI功能再考虑Pro：免费版本身已经是一个完整的编辑器了，AI是额外的加成
  • 关注插件更新：生态在逐步完善，常用的插件可能很快就会出现

适合谁用？

推荐：

  • 追求编辑器速度和响应体验的开发者
  • 对AI编程辅助有需求的个人开发者
  • 小团队需要轻量级协作工具的

可考虑：

  • 习惯VS Code插件生态的用户（需要确认常用插件是否有替代）
  • Windows用户（可以先用免费版测试稳定性）

不推荐：

  • 对插件生态依赖极高，暂时离不开VS Code的
  • 需要完整企业级功能（如SSO、详细权限管理）的团队

❓ 常见问题

Zed Editor是免费的吗？

基础版本完全免费，包含核心编辑功能和基础AI辅助。Pro版本每月$20，解锁高级AI功能和协作功能。

Zed Editor和VS Code哪个好？

Zed在启动速度和响应性能上优于VS Code，适合追求极致效率的开发者。VS Code插件生态更丰富，企业支持更好。选择取决于具体需求。

Zed Editor支持哪些AI模型？

Zed AI支持Claude 3.5、Gemini Pro、GPT-4o等多种模型，用户可根据需求切换或自定义API Key。

Zed Editor适合新手吗？

Zed Editor界面简洁直观，对编程新手友好。但考虑到插件生态，建议有一定编程基础的用户使用更能发挥其优势。
