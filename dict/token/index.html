Token (标记)

Token
基础概念
AI基础计费

Token是AI模型处理和计费的基本单位。理解Token对于控制API成本、优化Prompt设计至关重要。

Token是什么

  • AI不直接处理"字"，而是将文本切成Token
  • 英文：1 token ≈ 0.75个单词（"artificial intelligence" = 2 tokens）
  • 中文：1 token ≈ 1-2个汉字（"人工智能" = 2-3 tokens）
  • 代码：1 token ≈ 1-4个字符

为什么重要

  • 成本计算：API按输入+输出Token总量计费
  • 上下文限制：每个模型有最大Token限制（如128K、1M），超出会被截断
  • 速度影响：输出Token越多，生成时间越长

2026年主流模型价格参考（每百万Token）

  • DeepSeek-V3：输入¥1，输出¥2
  • GPT-5：输入$2.5，输出$10
  • Claude 4：输入$3，输出$15
  • Gemini 2.5 Flash：输入$0.15，输出$0.6

省钱技巧

精简System Prompt、使用更短的示例、选择合适的模型规格（小模型处理简单任务）。
